# x402-axios Library Documentation

## Overview

The `x402-axios` library is a specialized HTTP client for making requests to endpoints protected by payment protocols. It extends the popular `axios` library with payment functionality, allowing applications to automatically handle payment requirements for API endpoints, particularly in the context of Web3/Solana applications.

The name "x402" refers to the HTTP status code "402 Payment Required", though the implementation provides a more comprehensive payment protocol for microtransactions in the Web3 ecosystem.

## Installation

```bash
npm install x402-axios
```

## Core Methods and Functions

### `withPaymentInterceptor(axiosInstance, signer)`

This function wraps an existing axios instance with payment interceptor functionality. It automatically handles the payment process when making requests to endpoints that require payment.

**Parameters:**
- `axiosInstance`: An existing axios instance to be wrapped
- `signer`: A signer object created using `createSigner` function

**Returns:**
- A new axios instance with payment interceptor capabilities

### `createSigner(network, privateKey)`

Creates a signer object for a specific blockchain network with the given private key. The signer is used to sign payment transactions.

**Parameters:**
- `network`: The blockchain network (e.g., "solana-devnet", "base-sepolia")
- `privateKey`: The private key for signing transactions

**Returns:**
- A signer object for the specified network

### `decodeXPaymentResponse(headerValue)`

Decodes the payment response from an x-payment-response header.

**Parameters:**
- `headerValue`: The value of the "x-payment-response" header from the HTTP response

**Returns:**
- Decoded payment information

### `MultiNetworkSigner`

A type/interface for handling multiple network signers simultaneously (EVM and SVM networks).

## Usage Examples

### Basic Usage

```typescript
import axios from "axios";
import { withPaymentInterceptor, decodeXPaymentResponse, createSigner, type Hex } from "x402-axios";

const privateKey = "0x..."; // Your private key
const baseURL = "https://api.example.com";
const endpointPath = "/premium-endpoint";

// Create a signer for the Solana devnet
const signer = await createSigner("solana-devnet", privateKey);

// Create an axios instance with the payment interceptor
const api = withPaymentInterceptor(
  axios.create({
    baseURL,
  }),
  signer,
);

// Make a payment-enabled request
const response = await api.get(endpointPath);
console.log(response.data);

// Decode the payment response information
const paymentResponse = decodeXPaymentResponse(response.headers["x-payment-response"]);
console.log(paymentResponse);
```

### Multi-Network Usage

```typescript
import axios from "axios";
import {
  withPaymentInterceptor,
  decodeXPaymentResponse,
  createSigner,
  type Hex,
  MultiNetworkSigner,
} from "x402-axios";

const evmPrivateKey = "0x...";
const svmPrivateKey = "..."; // Solana private key

// Create signers for different networks
const evmSigner = await createSigner("base-sepolia", evmPrivateKey);
const svmSigner = await createSigner("solana-devnet", svmPrivateKey);
const signer = { evm: evmSigner, svm: svmSigner } as MultiNetworkSigner;

// Use with payment interceptor
const api = withPaymentInterceptor(
  axios.create({
    baseURL: "https://api.example.com",
  }),
  signer,
);

const response = await api.get("/premium-endpoint");
console.log(response.data);

const paymentResponse = decodeXPaymentResponse(response.headers["x-payment-response"]);
console.log(paymentResponse);
```

### Integration with Solana Agent Kit

```typescript
import axios from "axios";
import { withPaymentInterceptor, decodeXPaymentResponse, createSigner, type Hex } from "x402-axios";
import { SolanaAgentKit } from "solana-agent-kit";

/**
 * Make a payment-enabled request using x402-axios
 */
export async function makeX402PaymentRequest(
  agent: SolanaAgentKit,
  baseURL: string,
  endpointPath: string
): Promise<{ data: any; paymentInfo: any }> {
  try {
    // Get the private key from the agent's wallet
    const privateKey = agent.wallet.getSecretKey() as unknown as Hex;
    
    // Create a signer using the agent's network
    const signer = await createSigner("solana-devnet", privateKey);
    
    // Create an axios instance with the payment interceptor
    const api = withPaymentInterceptor(
      axios.create({
        baseURL,
      }),
      signer,
    );
    
    // Make the request
    const response = await api.get(endpointPath);
    
    // Decode the payment response
    const paymentResponse = decodeXPaymentResponse(response.headers["x-payment-response"]);
    
    return {
      data: response.data,
      paymentInfo: paymentResponse
    };
  } catch (error) {
    console.error("Error making payment request:", error);
    throw new Error(`Payment request failed: ${error instanceof Error ? error.message : 'Unknown error'}`);
  }
}

/**
 * Get payment information for a resource without fetching the full resource
 */
export async function getX402PaymentInfo(
  agent: SolanaAgentKit,
  baseURL: string,
  endpointPath: string
): Promise<any> {
  try {
    // Get the private key from the agent's wallet
    const privateKey = agent.wallet.getSecretKey() as unknown as Hex;
    
    // Create a signer using the agent's network
    const signer = await createSigner("solana-devnet", privateKey);
    
    // Create an axios instance with the payment interceptor
    const api = withPaymentInterceptor(
      axios.create({
        baseURL,
      }),
      signer,
    );
    
    // Make a HEAD request to get payment information without fetching the full resource
    const response = await api.head(endpointPath);
    
    // Decode the payment response
    const paymentResponse = decodeXPaymentResponse(response.headers["x-payment-response"]);
    
    return paymentResponse;
  } catch (error) {
    console.error("Error getting payment info:", error);
    throw new Error(`Failed to get payment info: ${error instanceof Error ? error.message : 'Unknown error'}`);
  }
}
```

## Configuration

To use x402-axios effectively, ensure you have:

1. A compatible private key for the target blockchain network
2. Sufficient funds in the associated wallet for making payments
3. An endpoint that supports the x402 payment protocol

## Environment Variables

When using in a project, you'll typically need these environment variables:

- `PRIVATE_KEY`: The private key of the signer
- `RESOURCE_SERVER_URL`: The URL of the resource server requiring payment
- `ENDPOINT_PATH`: The path of the endpoint to call on the resource server

## Error Handling

The library follows standard axios error handling patterns. Payment-related errors will surface through the usual axios error mechanisms, with additional details in the response headers when available.
